Image Corruption II

Progress!

So I may have lied a bit. This post will not be the all-inclusive tutorial I promised on image corruption. This is more of a status update. I do have a working prototype of the corruption script, which I'm tentatively calling pyCorrupt (how creative...) As you can probably guess, the script is written in python. This is for a couple of reasons.

  1. Python is convenient.
  2. I'll probably be using flask to host the polished project.

The script at the moment, is hardcoded, hacky, and altogether unpleasant to look at, however it is functional. As of right now, the script only supports .jpg images, however I'll probably look into making it work with gifs, provided I can find the proper information regarding its file headers (I'll go into this in much more detail in my full post)

So far here have been the few key things that I've taken from this project

  • .jpg files are split by markers (the only marker important to us in this context is Start of Scan (SOS)
  • Hex values can for the most part be changed freely between the SOS and End of File (EOI) marker, without completely breaking the image. I have noticed in some cases that changing hex values right after the SOS marker will break an image, however I'll have to look into why this happens.
